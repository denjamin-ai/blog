# 3. Bible Without Structure

**Problem**: Long prompt without structure, important content drowns in text.

**Anti-pattern**:
- 10 screens of text without headers
- Critical rules mixed with stylistic guidelines

**Countermeasures**:
- Structure like RFC:
  1. Critical rules (at the beginning)
  2. Role and specialization
  3. Instruction priorities
  4. Workflow
  5. Output format
  6. Examples
- XML tags to separate sections
- Secondary content at the end
