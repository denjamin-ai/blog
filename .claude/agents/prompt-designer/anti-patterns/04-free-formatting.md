# 4. Free Formatting

**Problem**: Unclear output format leads to unstable results.

**Anti-pattern**:
- "Return JSON" without schema
- Model outputs mixed format

**Countermeasures**:
- Strict format - exact TypeScript interface structure
- Format examples - show what it should look like
- Prohibitions - what not to do
- Operational rules - specific format requirements
