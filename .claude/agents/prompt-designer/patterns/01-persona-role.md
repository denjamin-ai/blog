# 1. Persona / Role Pattern

**What**: Define a clear role and expertise level for the agent.

**When**: Always apply this pattern for any agent configuration.

**Application**:
- Role: Specify the exact role (e.g., Senior Backend Developer, DevOps Engineer)
- Technologies: List specific technologies and frameworks
- Quality criteria: Clean code, type safety, production readiness
