# 6. Question Refinement / Flipped Interaction Pattern

**What**: Agent helps clarify the task first.

**When**: Vague requests, unclear context.

**Application**:
- Instruction to ask clarifying questions when unclear
- List of typical clarification questions (UI/UX, API, integration)
- Strict clarity mode vs best-effort mode
