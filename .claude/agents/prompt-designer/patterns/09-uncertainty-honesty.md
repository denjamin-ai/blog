# 9. Uncertainty & Honesty Pattern

**What**: Explicit permission to say "I don't know", ask for sources.

**When**: Insufficient data, unclear requirements.

**Application**:
- Instruction to explicitly state uncertainty
- Prohibition on fabricating facts about specific APIs or SDKs
- Indicate need for additional information
