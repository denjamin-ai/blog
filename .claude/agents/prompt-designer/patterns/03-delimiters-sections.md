# 3. Delimiters & Sections Pattern

**What**: Structure prompts with explicit delimiters (XML/markdown).

**When**: To separate instructions, data, and examples.

**Application**:
- Use XML tags: `<ARCHITECTURE>`, `<GOAL>`, `<CONSTRAINTS>`, `<WORKFLOW>`
- Section separation: system rules, goal, constraints, workflow
- Prompt injection protection: clear separation of instructions and data
