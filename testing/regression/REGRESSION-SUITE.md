# Regression Test Suite

**Цель:** Полная проверка существующей функциональности перед релизом  
**Время выполнения:** ~3–4 часа (полный) / ~60 мин (targeted)  
**Частота:** Перед каждым релизом; после крупных изменений

---

## Execution Order

```
1. Smoke Suite (15 мин) — если провал, стоп
2. P0 Security тесты (20 мин) — если провал, стоп
3. P0 Critical path тесты (40 мин)
4. P1 Major features (90 мин)
5. P2 Edge cases (60 мин)
6. Exploratory (30 мин)
```

---

## Pass/Fail Criteria

| Уровень | Критерий |
|---------|---------|
| **PASS** | Все P0 прошли; ≥ 90% P1 прошли; нет критических открытых багов |
| **FAIL (блок релиза)** | Любой P0 провалился; критический баг без workaround |
| **CONDITIONAL** | P1 провалы с документированным workaround |
